# 文献检索与登记

目标是找到符合项目范围、身份可靠且值得用户判断的候选论文，并通过服务器登记。

## 工作方式

1. 从工作区选择现有项目并明确范围、时间和停止条件。
2. 自主组合检索式和来源；优先论文官方页、出版社、arXiv、OpenReview 等可核验来源。
3. 核对标题、作者和至少一个稳定公开标识符；有歧义时不要静默合并。
4. 从 `GET /api/schema/paper` 获取当前提交结构。
5. 批量提交论文事实和项目判断。新候选使用 `discovered`。
6. 回读项目论文，报告新建、复用、未变化、冲突和信息缺口。

## 质量边界

- 作者按正式顺序保存完整名单；来源只显示 `et al.` 时标记不完整。
- 摘要、场所和年份无法可靠核验时允许为空。
- 不为满足字段完整度而编造中文标题、摘要、贡献或关系。
- 只有用户明确决定后才改变筛选状态。
- 收录至少需要可靠论文身份和对当前项目的相关性说明。

完成条件是目标候选已通过公开 API 登记或复用，身份冲突与缺口已明确，并且最终状态已经回读。
