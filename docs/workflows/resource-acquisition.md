# 论文资源获取

目标是为已登记论文获得合法、完整、身份一致且来源清楚的 PDF 或 TeX，并最终提供可阅读 PDF。

## 工作方式

1. 确认论文已经登记，读取现有资源和工作区能力。
2. 已有正确资源时复用，不重复上传。
3. 优先获取官方或作者提供的 PDF；记录来源 URL 和 origin。
4. 没有合适 PDF 但有完整 TeX 时，上传 zip、tar 或 tar.gz 源码包。
5. `compile.ready=true` 时可以提交编译任务；否则报告工具未就绪，不自动安装。
6. 轮询任务并回读输出资源，确认 PDF 可打开且派生关系存在。
7. 清理调用方创建的网络临时文件。

## 质量边界

- 文件必须与论文身份一致，不能用摘要页、登录页或损坏文件代替。
- TeX 包必须包含构建所需源码和依赖，不能包含越界路径、链接逃逸或异常压缩内容。
- 新文件创建新 Resource；切换首选不删除历史。
- 翻译只由用户明确触发，输入是首选原始 PDF，无论它来自下载还是 TeX 编译。

完成条件是资源和任务都已回读；成功时默认阅读 PDF 可访问，失败时原有首选资源保持不变且错误已报告。
