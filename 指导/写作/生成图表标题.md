# 生成图表标题

## 任务目标

把中文图表描述转为简洁、规范的英文 Figure 或 Table 标题。

标题应帮助读者识别对象、比较和条件，不应把未经证明的结论写进标题。若用户实际需要完整 caption，应先确认，因为标题与解释性图注承担不同责任。

## 输入要求

- 类型：Figure 或 Table。
- 图表对象、比较内容、数据集/条件和必要缩写。
- 希望使用名词短语还是完整句。
- 若已有图号、标签或 Tex 命令，明确是否需要保留。

## 生成规则

- 名词短语使用 Title Case 且末尾不加句号；完整句子使用 Sentence case 且加句号。
- 删除 `The figure shows`、`This diagram illustrates` 等冗余开头。
- Table 优先使用 `Comparison with`、`Ablation study on`、`Results on` 等准确表达。
- 不添加 `Figure 1:` 或 `Table 1:` 前缀。
- 转义 Tex 特殊字符，保留公式。
- 只在数据确实支持时使用 `effectiveness`、`improvement` 或 `superiority`。
- 标题中缩写应已在正文定义；否则使用全称或提醒用户。

## 常见陷阱

- `Results` 没说明在哪个数据集、指标或条件。
- 标题提前解释因果，例如 “Why Module A Improves Robustness”。
- Figure 与 Table 使用不一致的大小写风格。
- 将完整 caption 压缩成无法理解的名词串。

## 示例

中文输入：“不同上下文长度下任务恢复率和 token 成本的权衡，折线图。”

可用标题：

> Recovery Rate–Token Cost Trade-off across Context Lengths

该标题描述比较，不宣称某方法一定更优。

## 输出

默认只输出英文标题文本。若输入缺少类型、比较对象或条件，先指出缺失项，不猜测。
