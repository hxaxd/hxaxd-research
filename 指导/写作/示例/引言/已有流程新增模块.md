# 流水线版本 3（在已有流水线上引入新模块）

`版本 3：建立在先前的流水线之上，引入一个新模块，并用引导图展示基本想法。`

## 适用场景

本文保留已有流程的大部分结构，只在一个明确失败点引入具有独立机制的新模块。若
所谓新模块只是封装、提示词改名或参数增加，不应使用此模式强调创新。

```latex
% 本文提出一种基于学习的 snake 算法，名为 deep snake，用于实时实例分割。

% 受已有方法 [21, 25] 启发，deep snake 以初始轮廓为输入，并通过回归每个顶点的偏移量来对其进行变形。

% 我们的创新是引入循环卷积，以便在轮廓上高效学习特征，如图 1 所示。

% 我们观察到，轮廓是一个由一串顶点以闭环方式连接而成的环图。由于每个顶点的度都相同，且都等于 2，因此我们可以在顶点特征上应用标准的一维卷积。

% 考虑到轮廓是周期性的，deep snake 引入了循环卷积，它表示一个非周期函数（一维卷积核）按照标准方式与一个周期函数（定义在轮廓上的特征）进行卷积。

% 循环卷积的卷积核不仅编码了每个顶点的特征，也编码了相邻顶点之间的关系。相比之下，通用 GCN 通过池化来聚合邻居顶点的信息。我们循环卷积中的核函数本质上是一种可学习的聚合函数，表达能力更强，因此比使用通用 GCN 的方法表现更好，这一点也由我们在第 5.2 节中的实验结果验证。
```

## 使用提醒

- 先准确承认继承了什么已有流程。
- 说明原流程的具体失败点和新模块的作用位置。
- 方法差异应能由消融或控制实验区分。
- 不把已有流水线整体重命名为本文框架。
