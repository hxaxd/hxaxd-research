# 相关工作

## 1. 责任

相关工作负责确定本文在已有知识中的真实位置。它说明最相近工作解决了什么、依赖什么机制或假设、在哪个与本文有关的条件下停止, 以及本文的差异究竟落在问题、信息边界、方法机制、优化目标还是适用范围。

写作以原始论文为依据。检索摘要可以提供线索, 关键定位判断必须核对足以支持它的原文内容。

## 2. 先建立比较关系

从`叙事定稿.md`读取最相近工作与真实差异, 再补充:

- 实验中实际使用的强基线
- 与核心机制最接近但任务不同的工作
- 与任务相同但解决路线不同的工作
- 为核心认识提供历史先例或反例的工作

每篇文献至少明确它在当前论文中的比较角色。文献数量服从定位需要。

## 3. 按比较问题分组

每个主题段落通常完成四项内容:

1. 定义本段讨论的路线或问题范围
2. 概括代表工作的共同做法和已解决部分
3. 说明与本文目标有关的假设、限制或未覆盖条件
4. 给出本文在机制、信息边界、优化对象或范围上的真实差异

```tex
\section{Related Work}
% 主题一: 路线定义 -> 代表工作 -> 相关边界 -> 本文关系
% 主题二: 路线定义 -> 代表工作 -> 相关边界 -> 本文关系
% 按定位需要继续
```

主题按研究问题或方法关系组织, 不按年份机械罗列。

## 4. 写最相近工作

最相近工作获得最充分的比较。对每项决定性工作写清:

- 它的核心问题和主张
- 方法实际做了什么
- 与本文共享的部分
- 差异发生在哪个条件、机制或目标
- 该差异是否进入实验中的直接比较

若本文延续或组合了已有思想, 直接承认继承关系, 再说明新增知识。真实重合不能通过换名、分散到多个小节或只讨论外围差异而弱化。

## 5. 与实验和引言保持一致

- 引言中的技术问题应能在相关工作中找到公平的已有路线说明
- 实验中的强基线应在相关工作中获得相称定位
- 相关工作声称的差异应由方法事实或实验事实支持
- 不可比的公开数字不用于证明本文优于对方
- 新颖性措辞与 R1/R2 已核验的撞车边界一致

正式稿件使用核验后的引用键和`references.bib`。内部研究文档中的网页链接只用于追溯, 对外稿件不保留本地文件路径。

## 6. 自检

- 每个主题是否服务一个明确比较判断
- 最相近工作是否比外围工作获得更精确的讨论
- 是否同时说明对方已解决部分和真实边界
- 本文差异是否落在可核验的研究内容, 而不是名称或措辞
- 文字定位与基线身份、实验协议是否一致
- 每个关键判断是否能够回到原始论文
- 相关工作结束后, 读者是否知道本文继承什么、改变什么、仍不覆盖什么
