# 英文 Tex 扩写

## 1. 做什么

只在原文已有信息允许的范围内微幅增加约 5 至 15 个英文单词, 显化必要前提, 因果或结论, 不注水, 不创造新事实

如果用户需要增加新论点, 文献, 实验解释或整段内容, 这不属于微扩写, 应返回章节写作或研究文档补证据

## 3. 执行

### 可扩写内容

- 补充原文明确蕴含但未写出的逻辑连接
- 补回为理解机制所必需的主语, 对象或限定条件
- 在已有上下文中显化输入, 输出或比较对象
- 必要时增加精准解释或自然连接词

### 保护项

- 不添加数据, 实验现象, 贡献或未经证据支持的推断
- 保留 Tex 命令, 公式, 引用和段落结构
- 不改变否定, 比较, 数量, 适用范围和 Claim 强度
- 不用空洞形容词和背景句填充字数

### 执行步骤

1. 标出原句已有但表达不完整的信息
2. 说明计划新增内容从哪条原文或上下文推出
3. 在目标字数内做最小扩写
4. 回译检查是否产生新命题

### 示例

原句:

> The module filters the context.

若上下文已明确目标是当前子任务, 可扩为:

> The module filters the context to retain information relevant to the current subtask.

若“相关”并未在原文或方法中定义, 则不能擅自加入

## 4. 产出

1. 扩写后的英文 Tex
2. 中文回译
3. 中文修改记录, 指出新增逻辑来自原文何处

新增内容无法从原文或已有证据推出时必须删除
