# 英文 Tex 润色

## 任务目标

修正英文 Tex 的语法, 标点, 冠词, 句法和非母语生硬表达, 提高严谨性, 清晰度和可读性, 但不改变技术内容

润色不是改写研究逻辑, 原文存在事实矛盾, Claim 过强或信息缺失时, 先指出问题,
不要通过更流畅的英文把它隐藏起来

## 修改重点

- 主谓一致, 冠词, 时态, 介词, 单复数和标点
- 非母语搭配, 歧义指代和过长句法
- 学术语域, 术语一致和并列结构
- 影响理解的被动语态或名词化

## 规则

- 使用正式但常见的学术词汇, 不堆砌辞藻
- 不使用缩写形式, 如 `it's`, `doesn't`
- 避免方法名或系统名的所有格, 优先使用名词修饰或 `of` 结构
- 保留领域缩写, Tex 命令, 公式, 引用和已有格式
- 不主动增加粗体, 斜体, 列表或新论断
- 原文已经准确自然时克制修改
- 保留数字, 限定, 比较对象和 Claim 强度

## 执行步骤

1. 先确认句子要表达的技术命题
2. 修复会影响含义的语法, 再处理自然度
3. 对较大句法调整进行回译
4. 检查 Tex 结构和术语表

## 常见陷阱

- 为自然度删掉 `under this setting`, `may` 等限定
- 将普通“改善”改成 `significantly outperforms`
- 用罕见词替代准确常用词
- 把两个逻辑不同的句子合并后产生因果关系

## 输出

1. 润色后的英文 Tex
2. 中文回译
3. 中文修改记录, 区分语法修复和表达优化
