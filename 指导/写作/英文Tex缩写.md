# 英文 Tex 缩写

## 任务目标

在不损失核心信息, 技术细节, 参数和限定条件的前提下, 微幅减少约 5 至 15 个英文单词

若用户需要章节级大幅压缩, 应先按段落责任删除低优先级内容, 不应反复对单句做微缩写

## 可压缩位置

- 将冗长从句压缩为短语
- 在更简洁时把被动语态改为主动语态
- 删除 `in order to` 等无信息填充词
- 合并重复主语, 已在近邻句明确的重复背景和同义强调

## 保护项

- 保留原有 Tex 命令, 公式, 引用和段落结构
- 不添加强调, 列表或破折号
- 保留数字, 参数, 否定, 比较对象, 适用范围和因果条件
- 不删除使 Claim 从局部变为普遍的限定词

## 执行步骤

1. 标出句中必须保留的技术槽位
2. 只删除不承担信息的词组
3. 计算缩写幅度
4. 回译并逐项核对技术槽位

## 示例

可将:

> We use the module in order to reduce the amount of redundant context.

压缩为:

> We use the module to reduce redundant context.

不能删除 `under the same token budget` 这类公平性限定

## 输出

1. 缩写后的英文 Tex
2. 中文回译
3. 中文修改记录

输出前核对所有参数和限定条件仍存在, 缩写幅度超过目标时撤回过度修改
