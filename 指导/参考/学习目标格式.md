# 学习目标.md 格式

定义 `学习目标.md` 的固定节和字段, 该文件由 L1 学习规划与阅读方案生成

## 固定节

### 一句话边界

领域的一句话定义——研究对象, 输入输出, 不包含的相邻问题

### 本轮学习重心

| 字段 | 说明 |
|---|---|
| 阶段 | L1 学习规划与阅读方案 |
| 领域名 | 用户确认的中文领域名 |
| 用户目标 | 系统入门 / 选题入口 / 补特定方向 / 其他 |
| 第一批策略 | 小步快跑 / 系统补全 / 其他 |
| 默认篇数 | 数字范围 |

### 包含内容

列出本轮明确包含的子方向, 范式, 系统, 评测或方法类型

### 暂不展开

列出故意排除的内容及排除原因

### 当前不确定点

列出尚未与用户确认的假设——已有基础, 时间预算, 阅读深度等, 每项单独一行

### 核验边界

| 字段 | 说明 |
|---|---|
| 核验日期 | YYYY-MM-DD |
| 检索对象 | 本文件的书目核验范围 |
| 优先来源 | arXiv / OpenReview / 出版方官方页面 / 项目主页 |
| 限制 | 当前核验层级, 如"仅书目和摘要, 未精读正文" |
