# 实验证明

`实验证明.md`是 R3 与 R4 共用的正式研究记录。R3 固定共同实验协议、分点证明责任和预定交付; R4 在同一文件中回填实际交付与当前判定。

分点主张是证明责任, 不是实验运行单元。共享运行、实验矩阵和结果可以同时服务多条主张。

## 固定节

产出文件必须严格遵循以下一级节骨架, Agent 不得自行增减一级节。共同实验协议可以按需要使用表格。

```markdown
# 实验证明

## 共同实验协议

固定数据或任务划分、泄漏边界、方法变体、模型角色、执行框架、预算口径、流程记忆冻结与迁移规则、冒烟测试/预跑/正式运行边界、重试与排除规则、统计单位、重复次数、聚合方法和成本口径。

可以在这里使用实验矩阵, 并标明每组共享运行服务哪些分点主张。只固定会影响研究结论的实验条件, 不规定代码结构、库、脚本、目录或并行实现。

## 分点证明方案

每条需要经验支持的分点主张沿用`叙事定稿.md`中的三级标题。

### C1 主张标题

#### 核心对比

说明哪些共享运行、指标、中间现象、消融或成本结果直接回答该主张。

#### 替代解释与控制

说明可能产生同样结果的关键替代解释, 以及用什么对照、消融、预算匹配或不变量控制加以区分。

#### 支持、削弱和推翻判据

预先说明什么结果支持主张, 什么结果缩小或削弱主张, 什么结果推翻主张, 以及什么情况下仍不能判定。
```
