# 文献证据库

`文献证据库.md`是研究项目内的文献身份索引, 用来说明每篇被引用文献自己是谁。

## 固定节

产出文件必须严格遵循以下骨架, Agent 绝对不能自行增减一级节。

```markdown
# 文献证据库

研究想法、叙事、实验证明等文档如果使用某篇文献, 在自己的句子后用`[文献:E1]`引用; 本文件中必须有对应的`## E1 文献标题`条目。多篇文献写作`[文献:E1; E2]`。

## E1 文献标题(简写原名)

- 文献信息:
    - 译名:
    - 作者:
    - 组织:
    - 链接(如果有):
    - 投稿状态:
- 核心主张:
- 主要方法:
- 主要结论:

## E2 文献标题

- 文献信息:
    - 译名:
    - 作者:
    - 组织:
    - 链接(如果有):
    - 投稿状态:
- 核心主张:
- 主要方法:
- 主要结论:
```
