# 研究想法

`研究想法.md`是 R1 的正式产物, 保存用户 Idea 的准确描述和四个审查坐标。两个一级节为后续读取提供稳定入口。

## 文档结构

```markdown
# 研究想法

## idea

用正式、自然的中文描述用户真正想研究的问题、方法、可能有效的原因和当前边界。保留决定 Idea 身份的关系, 外部事实在使用处引用。

## 核心主张

### 共识

当前研究以什么背景判断为出发点。

### 问题

什么对象在什么场景或条件下出现什么问题, 产生什么后果。

### 方法

用户提出什么机制或系统变化来回应问题。

### 原因

这项变化为什么可能改善问题, 预期改变什么中间行为或可观察结果。
```
