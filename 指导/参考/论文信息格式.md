# 论文信息.md 格式

定义论文目录下 `论文信息.md` 的固定字段, 该文件由论文归档与 PDF 翻译能力在归档时生成

## 字段表

| 字段 | 说明 |
|---|---|
| 英文标题 | 官方标题 |
| 中文标题 | 翻译标题 |
| 作者 | 完整作者列表 |
| 年份 | 首次公开年份 |
| 发表状态 | 预印本 / 会议/期刊名 + 年份 + track |
| 稳定标识符 | DOI / arXiv ID / OpenReview forum ID |
| 版本 | 正式版 / 作者接受稿 / 预印本 vN |
| 来源 | 官方页面 URL |
| 归档日期 | YYYY-MM-DD |
| 文件 | 原文.pdf |
| SHA-256 | 原文 PDF 的 SHA-256 哈希 |
| 其他版本 | 如有多个版本, 记录差异和文件名 |
| 备注 | 可选, 记录特殊说明 |
