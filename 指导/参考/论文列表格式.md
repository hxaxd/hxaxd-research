# 论文列表.md 格式

定义 `论文列表.md` 的固定结构, 该文件由文献检索与归档在用户确认后生成

## 固定节

### 检索边界

说明本次检索的方向, 侧重和未覆盖内容

### 论文列表

| 顺序 | 论文 | 角色 | 选入原因 |
| --- | --- | --- | --- |
| 1 | 英文标题, 作者, 年份/发表 | 综述 / 奠基 / 方法 / 系统 / Benchmark / 反例 | 它在知识结构中承担什么不可替代的角色 |

### 候补池 (可选) 

适合后续追加的论文:

| 论文 | 角色 | 当前判断 |
| --- | --- | --- |
| 英文标题, 作者, 年份/发表 | — | 为什么不放第一批, 什么时候适合加入 |

### 推荐阅读顺序

分阶段说明阅读路径, 每阶段写明阅读目标

### 论文完整记录

#### <论文的中文短标题>

- 字段 1: 说明
- 字段 2: 说明
- 字段 3:
 - 说明 a
 - 说明 b
 - 说明 c

以下是所有需要字段:

| 字段 | 说明 |
| --- | --- |
| 英文标题 | 官方标题 |
| 中文标题 | 翻译标题 |
| 作者 | 第一作者 et al. |
| 组织 | 第一作者所属机构 |
| 时间 | 首次公开年份 |
| 发表状态 | 预印本 / 会议/期刊名 + 年份 |
| 类型 | 综述 / 奠基 / 方法 / 系统 / Benchmark / 反例 / 相邻工作 |
| 主要方法 | 一句话 |
| 贡献 | 核心贡献 |
| 选入原因 | 不可替代的角色 |
| 阅读重点 | 应关注哪些节/实验/论证 |
| 与其他论文关系 | 延续 / 反驳 / 扩展 / 提供评测 |
| 稳定链接 | DOI / arXiv / OpenReview |
| 源码/Demo | 可选, 文章源码仓库或 Demo 链接 |
| 官网 | 可选, 项目官网 |

## 篇数范围

- 宽领域: 12–25 篇
- 中等方向: 8–18 篇
- 具体问题: 6–12 篇
- 最小闭环: 5–8 篇

超出范围时按优先级分层
