# 论文处理记录.md 格式

定义论文目录下 `论文处理记录.md` 的固定字段, 该文件由论文归档与 PDF 翻译能力在每次处理时追加记录

## 字段表

每次处理, 归档, 翻译, 重新处理, 追加一条记录:

| 字段 | 说明 |
|---|---|
| 日期 | YYYY-MM-DD |
| 操作 | 归档 / 翻译 / 重新翻译 |
| 工具 | pdf2zh-next |
| 工具版本 | 固定版本号 |
| 命令 | 完整命令, 不含密钥 |
| 服务 | DeepSeek V4 Flash |
| 输入文件 | 输入 PDF 路径 |
| 输出文件 | 中文译文.pdf / 双语对照.pdf 路径 |
| 输出 SHA-256 | 各输出文件的哈希 |
| 退出状态 | 成功 / 失败 |
| 错误摘要 | 失败时的错误信息 |
| 备注 | 可选 |

## 追加规则

- 每次处理追加一条, 不覆盖历史记录
- 翻译失败时不写入输出文件路径和哈希, 只记录错误
- 机器验收不等于语义质量审校——在备注中明确此边界
