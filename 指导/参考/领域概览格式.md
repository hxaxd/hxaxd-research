# 领域概览

`领域概览.md`是领域目录内的知识结构索引, 由领域概览能力生成或更新。

## 固定节

产出文件必须严格遵循以下骨架, Agent 绝对不能自行增删节或改变节序。

```markdown
# <领域名> 领域概览

## 1. 这个领域在研究什么

## 2. 为什么这个领域重要

## 3. 主要子方向与技术路线

### 3.X <子方向名>

**定义**: <一句话定义>

**技术演化**:
- <旧限制> → <论文A> 提出 <新路线> → <论文B> 在此基础上 <延续/扩展/反驳/提供评测>
- ...

**边界**: <什么不算在内>

## 4. 当前发展状态与关键问题

### 4.1 已解决的问题

<方案成熟, 对应哪些 Benchmark 已饱和或接近饱和>

### 4.2 稳步推进的问题

<范式明确, 在现有 Benchmark 上持续改进, 对应哪些活跃 Benchmark>

### 4.3 仍在探索的问题

<尚无公认范式或评价方式本身还在形成中, 对应哪些早期 Benchmark 或评测尝试>
```
