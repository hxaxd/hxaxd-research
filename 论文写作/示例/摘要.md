```tex
\section{摘要}
```

## 版本二: 技术问题 -> 核心认知 -> 技术贡献

```tex
\section{摘要}
% 任务
%% 例子 1: 近年来, 生成模型因扩散模型的成功而取得了显著进展
%% 例子 2: 本文研究如何仅根据极少量相机视角为人体表演者合成新视角图像

% 已有方法面临的技术问题
%% 例子 1: 这些模型的成功通常归因于分类器引导和无分类器引导等技术, 它们为权衡保真度与多样性提供了有效机制. 但是, 这些方法无法引导生成图像感知自身的几何构型, 例如深度, 因而限制了扩散模型在需要深度感知的领域中的应用
%% 例子 2: 近期一些工作表明, 在输入视角密集时, 学习三维场景的隐式神经表示可以获得出色的新视角合成质量. 但是, 当视角极其稀疏时, 这种表示学习将成为病态问题

% 用一句话介绍解决技术问题的核心认知
%% 例子 1: 为了解决这一局限, 我们提出一种新的扩散模型引导方法, 使用从扩散模型丰富的中间表示中估计出的深度信息
%% 例子 2: 为了解决这一病态问题, 我们的核心思路是整合不同视频帧中的观测信息

% 用一到两句话介绍实现核心认知的技术贡献
%% 例子 1: 为此, 我们首先提出一个使用扩散模型内部表示的标签高效深度估计框架. 在采样阶段, 我们使用两种引导技术, 根据估计的深度图对生成图像进行自条件控制, 第一种采用伪标签, 第二种采用深度域扩散先验
%% 例子 2: 为此, 我们提出 Neural Body, 一种新的人体表示. 它假设不同帧中学习到的神经表示共享同一组锚定在可变形网格上的潜在编码

% 介绍技术创新的好处
%% 例子 2: 因此, 不同帧中的观测可以被自然整合. 可变形网格还为网络提供几何引导, 使其能够更高效地学习三维表示

% 实验
```

```tex
% 用一句话介绍技术贡献和技术优势
%% deep snake 使用神经网络反复变形初始轮廓, 使其与物体边界匹配, 从而以基于学习的方法实现经典 snake 算法的思想

% 用一句话介绍技术贡献和技术优势
%% 为了学习轮廓上的结构化特征, 我们在 deep snake 中采用循环卷积. 与通用图卷积相比, 循环卷积能够更好地利用轮廓的环图结构

% 用一句话介绍技术贡献和技术优势
%% 基于 deep snake, 我们构建了一个包含初始轮廓生成和轮廓变形的两阶段实例分割流程, 它能够处理物体定位误差

```
