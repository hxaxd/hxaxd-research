# 相关工作示例

## 版本一: 传统流程 -> 近期改进 -> 剩余瓶颈 ([DUSt3R](https://openaccess.thecvf.com/content/CVPR2024/html/Wang_DUSt3R_Geometric_3D_Vision_Made_Easy_CVPR_2024_paper.html))

```tex
\paragraph{运动恢复结构}
运动恢复结构 [18, 19, 40, 42, 83] 旨在根据一组图像重建稀疏三维地图, 同时确定相机参数
传统流程首先根据多幅图像之间的关键点匹配 [4, 5, 39, 53, 80] 获得像素对应关系并确定几何关系, 随后使用光束法平差联合优化三维坐标和相机参数
近年来, 运动恢复结构流程中的多个子过程都因学习式技术的引入而得到显著改进, 包括更强的特征描述 [23, 26, 77, 101, 127], 更准确的图像匹配 [3, 15, 27, 28, 51, 65, 81, 92, 96, 107], 基于特征度量的细化 [50] 和神经光束法平差 [49, 116]
尽管各个组成部分都取得了进展, 运动恢复结构仍然保留顺序执行的流程, 因而容易受到任一组件中噪声和误差的影响
```

## 版本二: 标准流程 -> 技术分支 -> 本文区别 ([ManhattanSDF](https://openaccess.thecvf.com/content/CVPR2022/html/Guo_Neural_3D_Scene_Reconstruction_With_the_Manhattan-World_Assumption_CVPR_2022_paper.html))

```tex
\paragraph{多视图立体重建}
许多方法采用两阶段流程完成多视图三维重建: 首先使用多视图立体重建估计每幅图像的深度图, 随后通过深度融合 [27, 31] 得到最终的重建结果
传统多视图立体重建方法 [43, 44] 能够恢复非常准确的三维形状, 并已用于新视角合成 [39, 40] 等多种下游任务; 但是, 这些方法通常在无纹理区域表现较差, 主要原因是无纹理区域中的稠密特征匹配难以完成
为了解决这一问题, 一些工作使用深度学习技术改进重建流程; 例如, [15, 52, 53] 提取图像特征并构建代价体, 再使用三维卷积神经网络预测深度图, [7, 13] 则以由粗到细的方式构建代价体并获得高分辨率结果
另一类工作 [11, 41, 48, 51] 使用场景先验辅助重建; 它们观察到, 可以利用平面先验补全无纹理平面区域, [19, 25, 56] 进一步提出深度与法向一致性损失来改善训练过程
与这些为每幅图像分别预测深度图的方法不同, 本文学习隐式神经表示, 从而获得更加一致和准确的重建结果
```
